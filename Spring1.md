## Sprint 1 概览
**Sprint目标**：为电动滑板车租赁服务建立完整的核心功能骨架，实现从用户注册、登录、查看车辆、租赁到归还的完整闭环流程，并提供基础的管理功能。在 Sprint 结束时，演示一个端到端可交互的系统。

## Sprint 待办事项列表
在此 Sprint 中计划完成以下产品待办事项。

### PBI-1：用户注册与登录 (US-1)
**用户故事**：
> As a 新用户，I want to 创建一个账户并安全登录，so that 我可以使用系统的核心服务。

**验收标准**：
*   前端提供包含（用户名、邮箱、密码、确认密码）的注册表单。
*   前端提供基于（邮箱/用户名和密码）的登录表单。
*   后端能够验证注册信息的完整性。
*   密码需使用哈希算法存储。
*   登录成功后端返回认证凭据（如 JWT Token）。
*   前端能够正确存储并管理用户的认证状态。

### PBI-2：查看可用滑板车 (US-2)
**用户故事**：
> As a 已登录用户，I want to 在一个界面（如地图或列表）上查看所有可用滑板车及其位置，so that 我可以选择一台滑板车进行租赁。

**验收标准**：
*   用户必须登录后才能访问滑板车查看页面。
*   页面仅显示状态为 “**available**” 的滑板车。
*   每条滑板车信息至少需包含：
    *   滑板车 ID
    *   型号
    *   当前电量
    *   当前位置
*   位置可以显示为地图上的标记点（如使用 Leaflet）或采用简单的文本列表形式。
*   界面应当能够直观地区分滑板车的**可用**与**不可用**状态。

### PBI-3：租赁滑板车 (US-3)
**用户故事**：
> As a 已登录用户，I want to 选择一台可用的滑板车并开始一个订单，so that 我可以使用这台滑板车。

**验收标准**：
*   在 PBI-2 实现的用户界面上为每辆可用滑板车提供 **“租赁”** 按钮。
*   点击按钮后，前端应弹出窗口以确认用户的租赁意向。
*   后端验证：
    *   用户身份的合法性
    *   滑板车当前状态的可用性
*   在数据库中成功创建租赁记录，并将租赁开始时间设置为当前时间。
*   将对应滑板车的状态更新为 **“in_use”**。
*   前端向用户显示租赁成功的提示信息。

### PBI-4：结束租赁 (US-4)
**用户故事**：
> As a 已登录用户，I want to 结束我正在进行的租赁订单，so that 我可以停止计费，并让滑板车恢复可用状态。

**验收标准**：
*   前端需提供明显的 **“结束租赁”** 按钮（可位于用户仪表板或独立页面）。
*   用户只能结束**自己当前正在进行**的租赁订单（即数据库中 `end_time` 字段为空的记录）。
*   后端能够计算租赁时⻓及费用（采用固定费率模型）。
*   更新对应租赁记录的结束时间与总费用。
*   将被租赁的滑板车状态恢复为 **“available”**。
*   前端向用户显示租赁结束及费用明细的确认信息。

### PBI-5：查看我的订单历史 (US-5) [备选]
**用户故事**：
> As a 已登录用户，I want to 查看我过去所有租赁的列表、详情及费用，so that 我可以追踪我的使用情况和消费。

**验收标准**：
*   前端提供 **“我的订单”** 专属页面。
*   显示用户所有历史租赁记录（包括已完成订单）。
*   每条记录需显示：
    *   租赁起始与结束时间
    *   所租赁滑板车的基本信息
    *   该次租赁的费用
*   所有数据默认按照租赁时间倒序排列。
*   用户能够点击查看单个订单的详细信息。

## Git 工作流说明
1.  **创建功能分支**：
    ```bash
    git checkout -b feature/US-1-auth
    # 或
    git checkout -b fix/booking-validation
    ```
2.  **小步频繁提交**：完成独立的小功能后及时 Commit，撰写清晰的提交信息。
3.  **推送分支**：
    ```bash
    git push origin feature/US-1-auth
    ```
4.  **创建 Pull Request**：在 GitHub 上创建 PR，详细描述变更内容，并关联相关 Issue。
5.  **代码审查**：至少需要 1 名其他团队成员审查并批准。
6.  **合并到主分支**：审查通过后合并，并删除功能分支。
